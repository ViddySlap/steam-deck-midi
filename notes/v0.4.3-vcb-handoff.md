### 3. Activate the 8 injected MIDI bindings (Resolume launch + UI tweak-revert)
The 8 deck → V-C-B bindings are already injected into `STEAMDECK V2.xml` (uniqueIds 1778280000001–8). When you launch Resolume they'll appear but be dormant. Activate each by opening Shortcuts → Edit MIDI on the V-C-B parameter, **changing the CC/note value to something else, then changing it back, and closing the dialog**. A click alone won't register them — the change-and-revert gesture is the trigger.

| CC 1 ch0 (L trigger pressure) | FLASH L2 AMOUNT |

| CC 2 ch0 (R trigger pressure) | FLASH R2 AMOUNT |

| Note 62 ch0 (R1 chaser) | CHASER R1 trigger |
| Note 61 ch0 (L1 flash) | FLASH L1 trigger |

**Bumper.Next is now bridge-driven, not MIDI-Learned.** `bumper_blast` was extended to handle CC 50 ch15 → Bumper.Next via REST PUT (chaser-layer R1 click → V-C-B emits CC 50 → bridge fires Bumper.Next). DECK_OUT/ch15 keys aren't decoded by the XML injection helper, so the bridge handles dispatch. Net effect for you: nothing to MIDI-Learn for Bumper.Next.

After tweak-reverting all 8, save the MIDI preset (or just close Resolume — it auto-saves the active preset).
