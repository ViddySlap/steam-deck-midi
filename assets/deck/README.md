# Deck Branding Assets

Drop Steam Deck launcher icons here.

## Expected Files

- `learn-map-icon.png`
  Used for the `Learn Steam Input Map` desktop launcher.

- `sender-icon.png`
  Used for the `STEAMDECK-MIDI-SENDER` desktop launcher.

- `install-wizard-icon.png`
  Used for the Deck installer bundle launcher.

## Recommendations

- Use PNG files.
- A square image around `512x512` works well for desktop launchers.
- Transparent backgrounds work best.

## Notes

- If these files are missing, the Deck installer falls back to generic system icons.
